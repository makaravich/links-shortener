A simple WordPress plugin for shortening links and tracking clicks.

== Description ==

Links Shortener is a lightweight WordPress plugin that allows you to create short links for any URL and track the number of clicks. It's perfect for social media sharing, affiliate marketing, or any situation where you need concise, trackable links.

Features:

Create short links for any URL

Track the number of clicks for each short link

Customizable short link

Easy-to-use interface integrated into the WordPress admin panel

== Installation ==

Upload the plugin files to the /wp-content/plugins/links-shortener directory, or install the plugin through the WordPress plugins screen directly.

Activate the plugin through the 'Plugins' screen in WordPress.

Use the Links Shortener menu item to create and manage short links.

== Frequently Asked Questions ==

= Can I customize the short link format? =

Yes, you can set a custom postfix for your short links in the plugin settings.

= What details will I be able to track when users click on links? =

Date and time, target URL, IP address, referrer, user agent, user language, user OS, user device type.
